# The Akademiya | Genshin Character Library 🌌

A responsive web application built with **Bootstrap 5** and **JavaScript** to manage and preview a collection of characters.

## 🛠 Features
- **Character Manager:** Track owned and unowned characters.
- **Dynamic Filtering:** Search by name or element.
- **Gacha Logic:** Simulates a "Summoning" animation to unlock new units.
- **Skill Previews:** Detailed descriptions and video demos (Embedded Media).

## 🎨 Software Design Principles
- **State Management:** Uses a centralized JavaScript object to drive the UI.
- **Conditional Rendering:** Grayscale filters for locked content.
- **Responsive Layout:** Mobile-first design using the Bootstrap grid.
